プライバシーポリシー

飲DX（以下「当社」）は、提供するシフト管理サービス「Shifty」（以下「本サービス」）における利用者の個人情報の取扱いについて、以下のとおりプライバシーポリシーを定めます。

1. 取得する情報

  • 店舗オーナー情報: 店舗名、氏名、メールアドレス、電話番号、住所
  • スタッフ情報: 氏名、時給、勤務時間、勤務希望、固定休
  • 業務情報: シフト編成内容、人件費、勤怠記録
  • 技術情報: IPアドレス、ブラウザ情報、Cookie、アクセスログ
  • 決済情報: クレジットカード情報は当社では保持せず、Stripe 社が PCI DSS 準拠の環境で管理します

2. 利用目的

  • 本サービスの提供・維持・改善
  • 利用料金の請求・収納
  • サポート・お問い合わせへの対応
  • サービスに関する重要なお知らせの送信
  • 新機能・キャンペーン情報の案内（オプトアウト可能）
  • 法令遵守・不正利用の防止

3. 第三者提供

当社は、利用者の同意なく個人情報を第三者に提供しません。ただし以下の場合を除きます：

  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要な場合
  • 業務委託先への提供（守秘義務契約締結のうえ）

4. 業務委託先

本サービスは以下の業務委託先のクラウドインフラを利用します：

  • Google Cloud Platform（Cloud Run / Firestore / Secret Manager）— インフラ・データ保管
  • Stripe, Inc. — 決済処理
  • Google Workspace — 業務メール

5. データ保管・セキュリティ

  • すべての通信は HTTPS で暗号化
  • データベース（Firestore）は Google Cloud により暗号化保管
  • パスワードは bcrypt によるハッシュ化保管（平文では一切保持しません）
  • セッショントークンは HTTP Only Cookie で保護
  • Content Security Policy 等のセキュリティヘッダを実装

6. データ保持期間

解約後 30 日間データを保持し、その後完全削除します。利用者からの削除請求があった場合は、法令で保持が義務付けられている情報を除き、速やかに削除します。

7. Cookie の利用

本サービスは利便性向上のためセッション Cookie を利用します。広告目的の Cookie は使用していません。

8. 開示・訂正・削除請求

個人情報の開示、訂正、削除、利用停止のご請求は support@in-dx.jp までご連絡ください。本人確認の上、合理的な期間内に対応します。

9. 改定

本ポリシーは法令の変更・サービス改善等に応じて改定する場合があります。重要な変更については本サービス上で告知します。
